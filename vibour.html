PCB Development

  • We find that people still suffer many problems when using the current navigation devices or mobiles for driving because:
  • 1. a device would repeat the same voice-reminding several times (noisy);
  • 2. voice-reminding still lacks the capability to provide clear routing-guide in some scenarios (misleading);
  • 3. drivers would move the sight away to the navigation screen (dangerous); and
  • 4. drivers cannot see the road at all during the night when vehicles moving towards (dangerous).
  • The device will be used as a physical plugin of mobile phones that users can easily set it to the glasses, so that users can get information from a transparent tiny-screen of the device, which displays navigational routing which perfectly overlaps with the road, in view from the eyes of the users. This structure provides a safety-driving environment because:â€‹
  • 1. transparent displaying will not block view sights (safe);

  • 3. duplicated voice reminding messages can be reduced (quiet driving environment); and
  • 4. the AR navigational routing can very distinctly and precisely points to the planned way (avoid ambiguous).
  • Compared with the AR glasses, this splitting-design also has many obvious advantages since:
  • 1. it is more compact with lightweight;
  • 2. it can be easily set onto or move away from the glasses;
  • 3. it has longer battery-usage time;
  • 4. it has lower heat generation near the brain; and
  • 5. it has lower electromagnetic radiation to the brain.
